Where to find documentation for [[https://wiki.contextgarden.net/Main_Page][ConTeXt]], a document production system
for typesetting documents.

# more

I had difficulty in finding documentation of this software, but no
longer.

All manuals and their TeX sources are shipped together with the
ConTeXt distribution itself! As of <2025-01-13 Mon>, one can go to
[[https://www.pragma-ade.nl/download-1.htm][ConTeXt download page]], download cont-lmt.zip, and obtain the
documentation by extracting doc subdirectory.

#+begin_src sh
7zz x cont-lmt.zip doc
#+end_src

In fact, of the 125MB ConTeXt LMTX distribution file, a whopping
~100MB of it are compiled PDF documentation, and a further 11MB are
TeX sources of the PDF documentation.  Only ~10MB are the actual LMTX
binaries.

A compressed tar archive of the
sources is available under [[https://1drv.ms/u/c/e8103a0f6490ccde/EQ6wPFtBX9lOidETaWs-ezgBIcuSgW5_d77Ba2fxVpMsjQ][Web/context-lmtx-doc-2024-12-29-22_39.tar.gz]].
This archive was produced with the following commands.

#+begin_src sh
find doc -type f -name '*pdf' -delete
tar acf context-lmtx-doc.tar.gz doc
#+end_src
